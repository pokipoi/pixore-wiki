# 清单规范

`manifest.json` 描述节点的元数据、接口和行为。本页是它的完整参考。

## 必需字段

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `name` | string | 在节点面板中显示的名称 |
| `category` | string | 分类路径（用 `/` 嵌套，如 `Image/Filters`） |
| `description` | string | 在 tooltip 与市场中展示的简短描述 |
| `version` | string | 语义化版本号（如 `1.0.0`） |
| `author` | string | 作者名或 GitHub 账号 |
| `inputs` | object | 输入端口定义 |
| `outputs` | object | 输出端口定义 |

## 输入端口规范

每个输入端口是一个对象：

```json
{
  "inputs": {
    "port_name": {
      "type": "IMAGE",
      "label": "Display Label",
      "default": null,
      "tooltip": "Optional tooltip text",
      "required": true,
      "options": {}
    }
  }
}
```

### 类型相关选项

**INT / FLOAT：**
```json
{
  "type": "INT",
  "label": "Width",
  "options": {
    "default": 512,
    "min": 1,
    "max": 4096,
    "step": 1
  }
}
```

**STRING：**
```json
{
  "type": "STRING",
  "label": "Prefix",
  "options": {
    "default": "img_",
    "multiline": false,
    "placeholder": "Enter prefix..."
  }
}
```

**BOOLEAN：**
```json
{
  "type": "BOOLEAN",
  "label": "Overwrite",
  "options": {
    "default": false
  }
}
```

**SELECT（下拉框）：**
```json
{
  "type": "STRING",
  "label": "Format",
  "options": {
    "default": "png",
    "choices": ["png", "jpg", "webp", "tiff"]
  }
}
```

## 输出端口规范

```json
{
  "outputs": {
    "port_name": {
      "type": "IMAGE",
      "label": "Output Image"
    }
  }
}
```

## 可选字段

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `icon` | string | 相对于节点目录的图标路径 |
| `tags` | string[] | 用于市场检索的标签 |
| `dependencies` | string[] | 必需的 pip 包 |
| `external_tools` | string[] | 需要的外部二进制（如 `["ffmpeg"]`） |
| `min_pixore_version` | string | 最低 Pixore 版本 |
| `license` | string | 节点协议（默认 MIT） |
| `repository` | string | 源码仓库 URL |
| `homepage` | string | 文档主页 URL |

## 完整示例

```json
{
  "name": "Smart Watermark",
  "category": "Image/Overlay",
  "description": "Add adaptive watermarks with position and opacity control",
  "version": "1.2.0",
  "author": "janedoe",
  "icon": "icon.png",
  "tags": ["watermark", "overlay", "branding"],
  "dependencies": ["opencv-python>=4.5"],
  "external_tools": [],
  "min_pixore_version": "1.0.0",
  "license": "MIT",
  "repository": "https://github.com/janedoe/pixore-watermark",
  "homepage": "https://github.com/janedoe/pixore-watermark#readme",
  "inputs": {
    "image": {
      "type": "IMAGE",
      "label": "Input Image",
      "required": true
    },
    "watermark": {
      "type": "IMAGE",
      "label": "Watermark Image",
      "required": true
    },
    "position": {
      "type": "STRING",
      "label": "Position",
      "required": true,
      "options": {
        "default": "bottom-right",
        "choices": [
          "top-left", "top-right",
          "bottom-left", "bottom-right",
          "center"
        ]
      }
    },
    "opacity": {
      "type": "FLOAT",
      "label": "Opacity",
      "required": false,
      "options": {
        "default": 0.5,
        "min": 0.0,
        "max": 1.0,
        "step": 0.05
      }
    }
  },
  "outputs": {
    "watermarked_image": {
      "type": "IMAGE",
      "label": "Watermarked Image"
    }
  }
}
```
