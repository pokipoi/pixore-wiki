# 安装

## 前置条件

- **Python 3.9** 或更新版本
- **Windows 10+**、**macOS 11+** 或现代 Linux 发行版
- 最低 **4 GB 内存**（运行 AI 节点和大批量任务建议 8 GB）
- 可选：支持 OpenCL 的 GPU 用于加速处理

## 快速安装

```bash
# 克隆仓库
git clone https://github.com/pokipoi/pixore.git
cd pixore

# 安装依赖
pip install -r requirements.txt

# 启动应用
python main.py
```

首次启动时，缺失的依赖会通过 `envinit.py` 自动安装。

## 预编译安装包

到 [Releases](https://github.com/pokipoi/pixore/releases) 页下载最新安装包。

| 平台 | 下载文件 | 状态 |
|----------|----------|--------|
| Windows  | `pixore-setup-x64.exe` | ✅ 可用 |
| macOS    | `pixore.dmg` | 🚧 规划中 |
| Linux    | `pixore.AppImage` | 🚧 规划中 |

## 开发者环境

如果你打算贡献代码或开发节点：

```bash
# 含子模块克隆（如有）
git clone --recurse-submodules https://github.com/pokipoi/pixore.git

# 以可编辑模式安装，附带开发依赖
pip install -e ".[dev]"

# 以详细日志模式启动
python main.py --debug
```

## 验证安装

启动后应看到带 Pixore Logo 的 **首页** 模块。试着打开左侧导航中的每个模块，确认都能正常加载。

## 故障排查

### Windows 上缺 DLL
安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)。

### PIL / Pillow 报错
```bash
pip uninstall pillow -y && pip install pillow
```

### macOS/Linux 提示权限不足
```bash
chmod +x bin/* && python main.py
```
