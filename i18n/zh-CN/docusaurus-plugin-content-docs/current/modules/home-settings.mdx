# 首页与设置

## 首页模块

启动时看到的着陆页。

### 最近文件
最近打开或处理过的图像缩略图网格。点击任意缩略图即可在相应模块中重新打开。

### 快捷操作
常用任务的一键入口：
- **快速压缩** —— 沿用上次设置压缩
- **批量缩放** —— 跳到缩放模块
- **自动切割** —— 用预设打开文件夹自动切割

### 动态封面
封面会根据使用习惯和时间动态变化。

### 彩带庆祝 🎉
大批量任务成功完成时，首页会洒下彩带 —— 为辛劳工作奉上一点小确幸。

## 设置模块

配置 Pixore 适配你的工作流。

### 外观
- **主题** —— 浅色、深色或跟随系统
- **强调色** —— 自定义 Fluent UI 强调色
- **字号** —— 缩放界面文字

### 性能

| 设置 | 说明 |
|---------|-------------|
| **最大进程数** | 并行 Worker 数量（默认 = CPU 核数） |
| **GPU 加速** | 在兼容操作中启用 OpenCL/CuPy |
| **缩略图缓存** | 缩略图 LRU 缓存大小限制 |
| **异步加载** | 启用非阻塞图像加载 |

### 路径

- **默认输入** —— 文件选择器的起始路径
- **默认输出** —— 处理结果保存位置
- **临时目录** —— 中间文件的暂存空间
- **缓存目录** —— 缩略图与模型缓存位置

### 工作流默认值

- **默认工作流类型** —— 管线 或 节点图
- **自动保存间隔** —— 自动保存频率
- **日志级别** —— 执行日志的详尽度

### 外部工具

如果自动检测失败，可手动配置内置工具路径：

- MozJPEG 二进制路径
- Guetzli 二进制路径
- pngquant 二进制路径
- ImageMagick 路径

### 通知

- **系统托盘** —— 启用/禁用托盘图标
- **任务完成** —— 任务结束时弹出通知
- **错误提醒** —— 出错时弹出通知
