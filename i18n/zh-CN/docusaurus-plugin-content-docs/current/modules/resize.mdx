# 批量缩放模块

生产级批量图像缩放工具，基于 **虚拟表格视图**，处理上千张图片也不会卡顿。

## 功能特性

### 预设
保存并调用常用的缩放配置。预设保存宽度、高度、算法和宽高比设置，方便快速复用。

### 智能预裁剪
通过专门的裁剪设置对话框，在缩放 **之前** 自动裁掉透明边缘，确保输出不会浪费像素在空白上。

### 多种算法

| 算法 | 适用场景 |
|-----------|----------|
| **Lanczos** | 高质量缩小（默认） |
| **Bicubic** | 通用缩放 |
| **Bilinear** | 速度优先 |
| **Nearest** | 像素艺术、复古图形 |

提供质量预览，帮助你直观选择算法。

### 并行处理
多进程池：
- 实时进度条
- 任意时刻可取消
- 拖拽时按扩展名过滤（只接受支持的图像格式）

### 预览模式
**表格视图**（详细信息）与 **网格视图**（缩略图）任意切换。

### 状态保留
应用重启后能恢复未完成的任务 —— 不会丢失工作。

## 使用步骤

1. 在导航栏打开 **批量缩放** 模块
2. 把图片拖入表格，或点击 **添加文件**
3. 设置目标尺寸与算法
4. 可选：在裁剪设置里启用 **智能预裁剪**
5. 点击 **处理** 开始批量缩放

## 快捷键

| 快捷键 | 作用 |
|----------|--------|
| `Ctrl+O` | 添加文件 |
| `Ctrl+Shift+O` | 添加文件夹 |
| `Ctrl+Enter` | 开始处理 |
| `Escape` | 取消处理 |
