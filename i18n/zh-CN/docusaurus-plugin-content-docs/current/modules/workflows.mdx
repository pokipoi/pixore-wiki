# 工作流模块

旗舰功能：一个 **完整的基于节点的可视化编程环境**，专为图像处理自动化打造。

## 工作流类型

### 管线模式
线性、顺序的流程，适用于简单自动化。

```
[文件输入] → [缩放] → [压缩] → [保存]
```

最适合直来直去、没有分支的处理流程。

### 节点图模式
带 **分支、汇合与拓扑执行** 的可视化画布。

```
              ┌→ [缩放 512] ─┐
[文件输入] ─┤                 ├→ [保存]
              └→ [缩放 1024] ┘
```

引擎会计算依赖关系，并行执行互不依赖的分支。

## 画布特性

### 无限画布
鼠标滚轮缩放，按住中键拖动平移，没有边界，节点想怎么排就怎么排。

### 网格吸附
节点对齐到隐形网格，整洁美观。`Ctrl+G` 切换。

### 智能走线
连线自动绕开节点，使用正交贝塞尔曲线。需要时也可手动控制。

### 小地图
角落的小地图概览整张图。点击即可跳转到任意位置。

## 执行

### 实时进度
每个节点显示其执行状态：
- ⏳ **等待中** —— 等待上游节点完成
- 🔄 **运行中** —— 正在处理
- ✅ **完成** —— 成功结束
- ❌ **错误** —— 失败（点击查看错误日志）

### 实时日志
执行面板里实时输出日志，可按节点或日志级别过滤。

### 取消支持
随时可停止执行，已处理的文件会保留。

## 节点市场

在内置应用商店浏览、安装、更新、评分社区节点。

- **浏览** —— 分类列表带搜索
- **安装** —— 一键下载并注册
- **更新** —— 检查已安装节点的更新
- **评分** —— 星级评分与评论
- **可配置索引** —— 可指向自建的远程索引

## AI 工作流生成

用自然语言描述工作流：

> "把所有 PNG 缩放到 512x512，用 pngquant 以质量 80 压缩，然后保存到输出目录"

AI 会自动生成完整的节点图。

## 触发器

文件系统监听器，自动启动工作流：

| 事件 | 触发条件 |
|-------|---------|
| **新文件** | 监听目录中创建文件 |
| **修改** | 已有文件被修改 |
| **删除** | 文件被移除 |

配合 **节流控制** 处理大批量文件。

## 工作流库

- **保存** —— 基于 JSON 的工作流文件
- **整理** —— 文件夹、标签、收藏
- **搜索** —— 跨工作流名称与描述全文检索
- **导入/导出** —— 与社区分享工作流

## 代码视图

查看任意工作流生成的执行代码。便于调试、学习节点工作机制，或将逻辑提取到 Pixore 之外使用。

## 快捷键

| 快捷键 | 作用 |
|----------|--------|
| `Space + 拖拽` | 平移画布 |
| `Ctrl+Z` / `Ctrl+Shift+Z` | 撤销 / 重做 |
| `Ctrl+G` | 切换网格吸附 |
| `Delete` | 删除选中的节点 |
| `Ctrl+A` | 全选 |
| `F` | 视图适配所有节点 |
| `Ctrl+S` | 保存工作流 |
| `F5` | 运行工作流 |
