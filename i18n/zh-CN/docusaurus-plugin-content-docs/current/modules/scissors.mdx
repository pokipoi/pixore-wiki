# 画布切割模块

灵感源自 Photoshop 的画布工具，用于 **精准图像切割** 与素材抽取。

## 绘图工具

| 工具 | 快捷键 | 说明 |
|------|----------|-------------|
| **选择器** | `V` | 移动并调整切割区域 |
| **钢笔** | `B` | 自由路径绘制 |
| **直线** | `L` | 直线切割 |
| **矩形** | `M` | 矩形区域选择 |
| **椭圆** | `O` | 椭圆区域选择 |
| **橡皮** | `E` | 移除切割边界 |

## 切割模式

### 手动切割
用绘图工具定义切割区域，然后将每个区域导出为单独的图片。

### 自动切割
检测 **连续色块** 并自动切片。

- `Ctrl+Shift+A` —— 对当前图片自动切割
- 最适合元素分明、背景一致的图像

### 批量自动切割
对 **整个文件夹** 应用自动切割，带进度对话框。

- `Ctrl+Alt+A` —— 打开批量自动切割对话框
- 选输入文件夹 → 选输出文件夹 → 让 Pixore 处理全部图片

## 工程文件（`.cut`）

把切割会话保存为 `.cut` 工程文件：

- 保留所有切割区域注解
- 保存工具设置与偏好
- 任何时候都能重新打开继续工作

## 导出选项

- **始终裁剪输出** —— 导出时自动裁掉透明边缘
- **格式选择** —— 导出为 PNG、JPEG、WebP 或 TIFF
- **智能导出** —— `Ctrl+Shift+E` 一键导出全部切割区域

## 标签页式工作区

可以同时在多个标签页中打开图像。每个标签页独立维护自己的切割区域和工具状态。

## 快捷键

| 快捷键 | 作用 |
|----------|--------|
| `Ctrl+Shift+A` | 自动切割当前图像 |
| `Ctrl+Alt+A` | 批量自动切割文件夹 |
| `Ctrl+Shift+E` | 导出所有切割区域 |
| `Ctrl+S` | 保存 `.cut` 工程 |
| `Ctrl+O` | 打开图像 |
| `Delete` | 移除选中的切割区域 |
