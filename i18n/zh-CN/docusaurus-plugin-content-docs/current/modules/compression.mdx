# 图像压缩模块

多引擎图像压缩工具，能 **自动为每种格式挑选最合适的工具**。

## 支持的引擎

### PNG 压缩 —— pngquant
基于质量目标的有损压缩（10–100 区间），通过削减调色板降低体积，同时保持视觉质量。

- **质量范围**：10（最小）到 100（无损回退）
- **自动回退**：达不到目标质量时自动回退到无损模式

### JPEG 压缩 —— MozJPEG
Google 出品的生产级 JPEG 编码器，在指定质量下追求最小体积。

- **质量范围**：1–100
- **适用场景**：每个字节都要省的网页图片
- **速度**：快，适合大批量

### JPEG 压缩 —— Guetzli
基于知觉质量优化，效果优秀但速度明显慢。

- **质量范围**：84–110
- **适用场景**：质量优先的最终交付素材
- **速度**：慢（每张图数分钟），不建议大批量使用

### 自动模式
指定一个 **最大文件大小（MB）**，Pixore 会搜索能满足体积预算的最高质量。

```
目标：500 KB
→ Pixore 尝试质量 90 → 620 KB ✗
→ Pixore 尝试质量 80 → 480 KB ✓
→ 结果：质量 80，体积 480 KB
```

### 各格式的处理管线

| 格式 | 引擎 | 选项 |
|--------|--------|---------|
| PNG | pngquant | 质量 10-100 |
| JPEG | MozJPEG / Guetzli | 质量、渐进式、优化 |
| WebP | PIL | 质量、无损模式、method |
| TIFF | PIL | 压缩类型、predictor |

## 输出选项

- **原地覆盖** —— 覆盖原文件（请谨慎使用）
- **输出文件夹** —— 保存到独立目录，原文件保留

## 使用步骤

1. 打开 **图像压缩** 模块
2. 为每种格式选择压缩引擎
3. 拖入文件或点击 **添加文件**
4. 选择输出模式（原地或文件夹）
5. 点击 **压缩**

使用 **自动模式** 时，请先设定目标文件大小再开始。
