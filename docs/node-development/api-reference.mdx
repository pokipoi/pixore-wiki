# Node API Reference

Reference for built-in utilities and helpers available to node authors.

## Image Utilities

Located in `common/image_utils.py`:

### `load_image(path) → PIL.Image`
Load an image from disk with automatic format detection.
```python
from common.image_utils import load_image
img = load_image("/path/to/image.png")
```

### `save_image(image, path, **kwargs) → str`
Save an image with format-specific options.
```python
save_image(img, "/output/image.jpg", quality=85, optimize=True)
```

### `get_image_info(image) → dict`
Extract metadata from an image.
```python
info = get_image_info(img)
# {"width": 1920, "height": 1080, "mode": "RGBA", "format": "PNG", "file_size": 2457600}
```

### `trim_transparent(image, threshold=0) → PIL.Image`
Auto-crop transparent borders.
```python
from common.image_utils import trim_transparent
cropped = trim_transparent(img, threshold=10)
```

### `resize_image(image, width, height, method="lanczos") → PIL.Image`
Resize with specified algorithm.
```python
resized = resize_image(img, 512, 512, method="lanczos")
# Methods: "lanczos", "bicubic", "bilinear", "nearest"
```

## Progress Reporting

Use the `payload` parameter to report progress:

```python
def process(self, image, ctx=None, payload=None):
    if payload:
        payload.update_progress(0.5, "Processing...")
    # ... do work ...
    if payload:
        payload.update_progress(1.0, "Done")
    return (image,)
```

### Payload Methods

| Method | Description |
|--------|-------------|
| `update_progress(ratio, message)` | Update progress bar (0.0 to 1.0) |
| `log(message, level)` | Write to execution log (`"info"`, `"warn"`, `"error"`) |
| `is_cancelled()` | Check if user cancelled execution |
| `set_result(key, value)` | Store execution result for code view |

## Compression Utilities

Located in `common/compress.py`:

```python
from common.compress import compress_png, compress_jpeg

# PNG compression via pngquant
compress_png("input.png", "output.png", quality=(60, 80))

# JPEG compression via MozJPEG
compress_jpeg("input.jpg", "output.jpg", quality=85)

# JPEG compression via Guetzli
compress_jpeg("input.jpg", "output.jpg", quality=95, engine="guetzli")
```

## File Utilities

Located in `common/file_utils.py`:

```python
from common.file_utils import collect_files, ensure_dir, get_unique_path

# Collect files matching patterns
files = collect_files("/input/folder", extensions=[".png", ".jpg", ".webp"])

# Ensure directory exists
ensure_dir("/output/folder")

# Get a unique filename (avoids overwrites)
path = get_unique_path("/output/image.png")  # → /output/image_1.png
```

## Batch Processing Pattern

For nodes that process multiple files, follow this pattern:

```python
import os
from common.image_utils import load_image, save_image
from common.file_utils import collect_files

class BatchProcessorNode:
    @classmethod
    def INPUT_TYPES(cls):
        return {
            "required": {
                "folder": ("FOLDER_PATH",),
                "width": ("INT", {"default": 512, "min": 1, "max": 4096}),
            },
        }

    RETURN_TYPES = ("FOLDER_PATH",)
    RETURN_NAMES = ("output_folder",)
    FUNCTION = "process"
    CATEGORY = "Image"

    def process(self, folder, width, ctx=None, payload=None):
        files = collect_files(folder, extensions=[".png", ".jpg", ".jpeg", ".webp"])
        output_dir = os.path.join(os.path.dirname(folder), "resized")
        os.makedirs(output_dir, exist_ok=True)

        for i, filepath in enumerate(files):
            if payload and payload.is_cancelled():
                break

            ratio = (i + 1) / len(files)
            if payload:
                payload.update_progress(ratio, f"Processing {os.path.basename(filepath)}")

            img = load_image(filepath)
            h = int(img.height * (width / img.width))
            resized = img.resize((width, h), Image.LANCZOS)
            save_image(resized, os.path.join(output_dir, os.path.basename(filepath)))

        return (output_dir,)


NODE_CLASS_MAPPINGS = {"batch_resizer": BatchProcessorNode}
```

## Error Handling

Return `None` for any output to signal failure. Log errors via payload:

```python
def process(self, image, ctx=None, payload=None):
    try:
        result = self.risky_operation(image)
    except Exception as e:
        if payload:
            payload.log(f"Error: {e}", "error")
        return (None,)  # Signal failure
    return (result,)
```
