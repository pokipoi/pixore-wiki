# Pixore

**An All-in-One Image Workstation with a Visual Workflow Engine**

Pixore is a desktop image-processing workstation that bundles **eight purpose-built tools** and a **node-based workflow engine** into a single Fluent-themed application.

From batch resizing and lossy/lossless compression to AI-assisted renaming, smart cutting, and fully programmable pipelines — Pixore replaces the dozen-or-so single-purpose utilities most creators juggle every day.

---

## Who Is This For?

- 🎮 **Game Developers** — Sprite atlas packing, asset pipeline automation, batch bit-depth conversion for retro art
- 📸 **Photographers** — Batch resize, multi-engine compression, auto-rename with reference style guides
- 🎨 **Content Creators** — AI-assisted background removal, intelligent file naming, smart canvas cutting
- 🌐 **Web Developers** — Image optimization pipelines, responsive image generation, automatic compression triggers
- ⚙️ **Power Users** — Chain AI services into reproducible pipelines, file-watcher automation, CLI-driven integrations

---

## Eight Modules at Your Fingertips

| Module | Purpose |
|--------|---------|
| **Home** | Recent files, quick actions, animated cover art, confetti celebrations |
| **Resize** | Batch image resizing with presets, smart pre-trimming, parallel processing |
| **Drag** | Drop files anywhere on the window for one-click quick processing |
| **Compression** | Multi-engine compression with MozJPEG, Guetzli, pngquant, auto-quality targeting |
| **Bitdepth** | Per-channel bit allocation, palettes, dithering presets |
| **Scissors** | Photoshop-style canvas cutter with drawing tools and batch auto-cut |
| **Renamer** | AI-driven batch renaming with multi-modal models and chat history |
| **Workflows** | Visual node-graph editor with marketplace, triggers, and AI generation |

---

## Key Highlights

- 🧩 **Modular Node System** — Every node is self-contained. Drop in new capabilities without touching the host.
- 🏪 **Built-in Marketplace** — Browse, install, update, and rate community nodes.
- 🤖 **AI-Powered** — Visual and language models for renaming, image generation, background removal, and node authoring.
- ⚡ **Trigger Automation** — File-system watchers launch workflows automatically.
- 🗜️ **Production Compression** — MozJPEG, Guetzli, pngquant, ImageMagick for best-in-class output.
- 🎨 **Modern Fluent UI** — Light/dark themes, frameless window, native drag-and-drop.
- 🔓 **Free & Open Source** — GPL v3 licensed.
