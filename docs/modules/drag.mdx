# Drag & Drop Module

A quick-action surface — drop files **anywhere on the window** for one-click processing. No navigation needed.

## How It Works

The Drag module provides a **transparent overlay** that accepts files dropped anywhere in the application window. Configure a default action, and every dropped file gets processed instantly.

## Features

### Always-On-Top Drop Zone
When enabled, the drop zone is always visible and accessible, even when working in other modules.

### Configurable Default Action
Choose what happens when files are dropped:

- **Quick Resize** — Resize to preset dimensions
- **Quick Compress** — Compress with saved settings
- **Quick Convert** — Convert to a target format
- **Run a Saved Workflow** — Execute a pre-built workflow on dropped files

### Notification Feedback
System tray notifications confirm successful processing. Click a notification to open the output folder.

### History
A rolling history of recently processed files with quick access to outputs.

## Usage

1. Open the **Drag** module
2. Select a default action from the dropdown
3. Drop files anywhere on the window
4. Processing starts immediately — results appear in the output folder

## Pro Tips

- Pin the Drag module as the **default launch page** in Settings for lightning-fast workflows
- Combine with **Triggers** for fully unattended automation
